Billing Codes

  • In order to add to "alotted" funny money you need to do a transaction
  • Can also add money with the Cashier (typically involves using real funds)
  • You can also automate the adding of funds
  • You can also add or deduct funds with a Scheduled Task
  • It is better to prompt for billing code at release time rather than at submission time because you don't have to use workstation client (which is costly)

Tech

Dillinger uses a number of open source projects to work properly:
